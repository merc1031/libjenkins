runJenkins :: ConnectInfo -> Jenkins a -> IO (Result HttpException a)Source

Query Jenkins API using Jenkins description

Successful result is either Disconnect or Value v

If HttpException was thrown by http-conduit, runJenkins catches it and wraps into Error. Other exceptions are not catched
